## File Structure

Each company folder contains **52 files**, including:
- **51 JSON service data files** - Accounting, financial, operational data
- **1 README.md file** - Documentation specific to that company's data

All files maintain the same structure as the original accounting sample data, but with company-specific customizations including:
- Company name references
- Industry-specific terminology
- Location details
- Relevant asset and product descriptions

## Usage

To use sample data for a specific company:

1. Select the company folder (e.g., `konkola-copper-mines-plc`)
2. The folder contains all necessary JSON files for testing and demonstration
3. Load the data from your application by referencing the specific company folder
